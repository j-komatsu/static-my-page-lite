# static-my-page-lite

シンプルなブックマーク管理ツール（軽量版）

## 概要

static-my-page-liteは、ブックマーク管理に特化した軽量版のマイページツールです。
元のstatic-my-pageから不要な機能を削除し、ブックマーク機能とページ追加機能のみを残しました。

## 特徴

- **軽量**: 不要な機能を削除し、シンプルで高速
- **ブックマーク管理**: 6つのセクションにリンクを整理
- **ヘッダーリンク**: よく使うリンクを最大5個まで表示
- **複数ページ**: プロジェクトごとに独立したページを作成可能
- **データ管理**: JSON形式でエクスポート/インポート可能
- **ドラッグ&ドロップ**: リンクの並び替えが簡単
- **1行表示モード**: コンパクトな表示が可能
- **通知なし**: 通知許可ダイアログが表示されません

## 削除された機能

元のstatic-my-pageから以下の機能を削除しました：

- TODOリスト機能
- メモ機能
- カレンダー機能
- ポモドーロタイマー（通知許可が不要になります）
- ナビゲーション設定機能

## 使い方

1. `index.html` をブラウザで開く
2. セクション名をクリックして編集
3. 編集ボタン（✏️）でリンクを追加・編集
4. ドラッグ&ドロップでリンクを並び替え
5. 必要に応じて新しいページを追加

## データの保存

- すべてのデータはブラウザのLocalStorageに保存されます
- データキーには `liteData_` プレフィックスが使用されます
- エクスポート機能でJSON形式でバックアップできます

## ファイル構成

```
├── css/

```

## ブラウザ対応

- Chrome, Edge, Firefox, Safari
- LocalStorageが使用可能なモダンブラウザ

## 開発者向け

### カスタマイズ

- `style.css`: デザインのカスタマイズ
- `script.js`: 機能の追加・変更
- 表示領域: max-width 1600px に拡大済み
- 余白: padding 10px に削減済み

### データモデル

LocalStorageキー：
- `liteData_{ページ名}`: セクションリンクデータ
- `liteNames_{ページ名}`: セクション名
- `liteTitle_{パス}`: メインタイトル
- `liteHeaderLinks_global`: ヘッダーリンク
- `liteProjects_data`: プロジェクト一覧
- `lite_started`: 開始フラグ
