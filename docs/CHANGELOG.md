## [2.0.0] - 2026-02-01

### 追加 - ポモドーロタイマー機能
- **基本機能**
  - 作業時間と休憩時間のカスタマイズ（1〜60分、1〜30分）
  - 開始/一時停止/リセット機能
  - セッション数のカウント
  - 自動継続モード（作業→休憩→作業を自動で繰り返す）

- **サウンド通知（3種類）**
  - 作業完了音: C6→E6→G6の上昇音（明るく達成感のある音）
  - 20%警告音: A5音を2回（注意喚起）
  - 休憩完了音: G6→E6の下降音（柔らかく優しい音）
  - Web Audio APIを使用したシンセサイザー音
  - サウンド通知のON/OFF設定

- **ブラウザ通知**
  - 作業完了時の通知（🍅 作業完了！）
  - 休憩完了時の通知（☕ 休憩完了！）
  - 残り20%時の警告通知（⚠️ 残り時間少！）
  - 通知のON/OFF設定
  - `requireInteraction: true` による持続的な通知表示

- **UI機能**
  - ヘッダー中央への配置
  - 最小化機能（−ボタン）
  - 完全非表示設定
  - タイマーの色変化（平常時: グレー、警告時: 黄色、休憩時: 緑色）
  - レスポンシブデザイン対応

- **状態管理**
  - LocalStorageへの状態保存（1秒ごと）
  - ページリロード時の状態復元
  - バックグラウンドでのタイマー動作

### 追加 - ナビゲーション設定機能
- ナビゲーションボタンの表示/非表示切り替え
- ナビゲーションボタンの並び替え（ドラッグ＆ドロップ）
- デフォルト設定へのリセット機能
- 設定のLocalStorageへの保存

### 変更
- ヘッダーレイアウトの改善
  - 左: ページタイトル
  - 中央: ポモドーロタイマー
  - 右: ヘッダーリンク
- モーダルダイアログのスタイル改善
- フッターにナビゲーション設定ボタンを追加

### 修正
- サウンド通知がOFFでもブラウザ通知音が鳴る問題を修正
  - `silent` オプションを設定に連動させるように修正
- 警告音が正しく再生されない問題を修正
  - 存在しない関数 `playPomodoroSound()` を `playWarningSound()` に修正
- 通知が視覚的に表示されない問題を修正
  - `requireInteraction: true` の追加
  - ユニークなタグの使用
  - 絵文字の追加で視認性向上

## [1.0.0] - 2025-XX-XX

### 追加
- **基本的なリンク管理機能**
  - カテゴリー別のリンク整理（6つのセクション）
  - リンクの追加・編集・削除
  - セクション名の編集
  - ドラッグ＆ドロップでの並び替え
  - 1行表示（コンパクト）モード

- **プロジェクト/ページ管理**
  - 複数のプロジェクトページ作成
  - ページごとに独立したデータ管理
  - ページの追加・削除機能
  - ページタイトルの編集

- **TODOリスト機能**
  - タスクの作成・完了・削除
  - 完了タスクの一括削除
  - LocalStorageでの永続化

- **メモ機能**
  - メモの作成・編集・削除
  - カテゴリー別管理
  - マークダウン対応（予定）

- **カレンダー機能**

- **ヘッダーリンク機能**

  - リンクの編集・削除

- **データ管理**
  - データのエクスポート（JSON形式）

  - LocalStorageへの自動保存

- **UI/UX**
