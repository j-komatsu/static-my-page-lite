## 1. リンク管理

### 概要
よく使うWebサイトやツールのリンクを整理して管理できます。カテゴリー（セクション）ごとに分類し、すばやくアクセスできます。

### セクションの編集

#### セクション名の変更
1. セクション名をクリック
2. テキストボックスが表示される
3. 新しい名前を入力
4. Enter キーまたは外側をクリックで保存

### リンクの管理

#### リンクの追加
1. セクション右上の **✏️ 編集ボタン** をクリック
2. モーダルが開く
3. 「リンクテキスト」と「リンクURL」を入力
4. （オプション）「1行表示（コンパクト）」をチェック
5. **リンクを追加** ボタンをクリック

#### リンクの編集
1. リンク一覧で編集したいリンクの **編集アイコン** をクリック
2. 単一編集モードに切り替わる
3. リンクテキストやURLを変更
4. **変更を保存** ボタンをクリック

#### リンクの削除
1. リンク一覧で削除したいリンクの **🗑️ 削除アイコン** をクリック
2. 確認ダイアログで「OK」を選択

#### リンクの並び替え
- リンクをドラッグ＆ドロップで並び替え可能

### 1行表示（コンパクト）モード
リンクを1行で表示することで、スペースを節約できます。多くのリンクを表示したい場合に便利です。

## 2. プロジェクト/ページ管理

### 概要
複数のプロジェクトやトピックごとに別々のページを作成できます。各ページは独立したリンク管理画面を持ちます。

### ページの追加

1. フッターの **ページ追加** ボタンをクリック
2. プロンプトでページ名を入力
3. 新しいページが作成され、ナビゲーションに追加される

### ページの削除

1. フッターの **ページ削除** ボタンをクリック
2. 削除したいページを選択
3. 確認ダイアログで「OK」を選択

**注意:** ページを削除すると、そのページのすべてのデータ（リンク、TODO、メモなど）が失われます。

### ページの切り替え

ヘッダーのナビゲーションボタンから、各ページに素早く移動できます。

## 3. TODOリスト

### 概要
タスクを管理し、完了状況を追跡できます。

### タスクの追加

1. TODOリストページに移動
2. タスク入力欄にタスク内容を入力
3. **追加** ボタンをクリック

### タスクの完了

- タスクの左側にあるチェックボックスをクリック
- 完了済みのタスクには取り消し線が表示される

### タスクの削除

#### 個別削除
- 各タスクの **削除** ボタンをクリック

#### 一括削除
- **完了タスクをすべて削除** ボタンをクリック
- 完了済みのタスクがすべて削除される

## 4. メモ機能

### 概要
テキストメモを作成・管理できます。マークダウンに対応しています。

### メモの作成

1. メモページに移動
2. **新規メモ** ボタンをクリック
3. タイトルと本文を入力
4. **保存** ボタンをクリック

### メモの編集

1. 編集したいメモをクリック
2. 内容を変更
3. **保存** ボタンをクリック

### メモの削除

1. 削除したいメモを選択
2. **削除** ボタンをクリック
3. 確認ダイアログで「OK」を選択

### カテゴリー管理

メモにカテゴリーを設定することで、整理しやすくなります。

## 5. カレンダー機能

### 概要
予定やイベントを日付ごとに管理できます。

### イベントの追加

1. カレンダーページに移動
2. 日付をクリック
3. イベント詳細を入力
4. **追加** ボタンをクリック

### イベントの表示

- カレンダー上で、イベントがある日付はハイライト表示される
- 日付をクリックすると、その日のイベント一覧が表示される

### イベントの削除

1. イベント一覧から削除したいイベントを選択
2. **削除** ボタンをクリック

## 6. ポモドーロタイマー

### 概要
ポモドーロテクニックを使用した時間管理ツール。作業時間と休憩時間を交互に繰り返すことで、集中力を維持します。

### 基本的な使い方

#### タイマーの開始
1. ヘッダー中央のポモドーロタイマーで **開始** ボタンをクリック
2. タイマーが作業時間のカウントダウンを開始
3. ボタンが **一時停止** に変わる

#### タイマーの一時停止
- **一時停止** ボタンをクリック
- ボタンが **再開** に変わる

#### タイマーのリセット
- **リセット** ボタンをクリック
- 現在のモードの初期時間に戻る

### タイマーの表示状態

#### 平常時（作業時間）
- グレーの枠線
- 白色の背景

#### 警告時（残り20%以下）
- 黄色の枠線
- 薄い黄色の背景
- タイマー数字がオレンジ色に変化

#### 休憩時間
- 緑色の枠線
- 薄い緑色の背景
- タイマー数字が緑色に変化

### セッション数
作業時間を完了するたびに、セッション数がカウントアップされます。

### 設定項目

ポモドーロタイマーは、⚙️（設定ボタン）から細かくカスタマイズできます。

#### 作業時間 (分)
- デフォルト: 25分
- 範囲: 1〜60分
- 1回の作業セッションの長さを設定

#### 休憩時間 (分)
- デフォルト: 5分
- 範囲: 1〜30分
- 作業後の休憩時間の長さを設定

#### 自動継続
- オン/オフ切り替え
- オンにすると、作業 → 休憩 → 作業を自動で繰り返す
- オフの場合、各フェーズ終了後に手動で開始が必要

#### サウンド通知（タイマー完了時）
- オン/オフ切り替え
- 作業完了時・休憩完了時に音が鳴る
- **音の種類:**
  - 作業完了: C6 → E6 → G6 の上昇音（明るく達成感のある音）
  - 休憩完了: G6 → E6 の下降音（柔らかく優しい音）

#### サウンド通知（残り20%時）
- オン/オフ切り替え
- 作業時間の残り20%になったときに警告音が鳴る
- **音の種類:** A5音を2回（注意喚起）

#### ブラウザ通知（アラート）を表示
- オン/オフ切り替え
- タイマー完了時と残り20%時にブラウザ通知を表示
- 初回有効化時、ブラウザの通知許可が必要

**通知の内容:**
- 作業完了: 🍅 作業完了！「作業時間が終了しました。休憩しましょう。」
- 休憩完了: ☕ 休憩完了！「休憩時間が終了しました。作業を再開しましょう。」
- 残り20%: ⚠️ 残り時間少！「作業時間の残りがX分を切りました。」

#### ポモドーロタイマーを表示する
- オン/オフ切り替え
- タイマー全体の表示/非表示を切り替え
- オフにすると、ヘッダーからタイマーが完全に消える

**非表示にしたタイマーを再表示する方法:**
1. フッターの **ナビゲーション設定** ボタンをクリック
2. ポモドーロタイマー設定を開く
3. 「ポモドーロタイマーを表示する」にチェックを入れる
4. **設定を保存** をクリック

### 最小化機能

タイマーを完全に非表示にせず、コンパクトに表示したい場合:

1. タイマーヘッダーの **−（最小化ボタン）** をクリック
2. タイマーの本体部分が非表示になり、ヘッダーのみ表示される
3. **□（展開ボタン）** をクリックで元に戻る

### サウンド通知とブラウザ通知の違い

| 項目 | サウンド通知 | ブラウザ通知 |
|------|------------|------------|
| **音** | Webオーディオで生成された音 | システムの通知音（設定OFFでサイレント） |
| **表示** | なし | デスクトップ通知が表示される |
| **設定** | アプリ内で完結 | ブラウザの許可が必要 |
| **用途** | 音だけで通知したい場合 | 他の作業中でも視覚的に通知がほしい場合 |

**推奨設定:**
- 集中作業中: サウンド通知 ON、ブラウザ通知 OFF
- マルチタスク時: サウンド通知 OFF、ブラウザ通知 ON
- 確実に気づきたい: 両方 ON

### ヒントとコツ

1. **初回使用時は標準設定（25分作業 / 5分休憩）から始める**
2. **自分に合った時間配分を見つける**
   - 集中力が続く時間は人それぞれ
   - 15分作業 / 3分休憩から始めても良い
3. **セッション数を記録する**
   - 1日に何セッション完了したか記録すると、生産性の目安になる
4. **休憩中は席を立つ**
   - ストレッチや水分補給で、次のセッションに備える

## 7. ナビゲーション設定

### 概要
ヘッダーのナビゲーションボタンをカスタマイズできます。

### ナビゲーション設定を開く

フッターの **ナビゲーション設定** ボタンをクリック

### ボタンの表示/非表示

各ボタンの左側のチェックボックスで、表示/非表示を切り替えられます。

**デフォルトで表示されるボタン:**
- MyPage
- TODOリスト
- メモ
- カレンダー
- ポモドーロタイマー設定

### ボタンの並び替え

1. 各ボタンの ⋮⋮ （ドラッグハンドル）をドラッグ
2. 好きな順序に並び替え
3. **設定を保存** ボタンをクリック

### デフォルトに戻す

**デフォルトに戻す** ボタンをクリックすると、初期設定に戻ります。

## 8. データ管理

### 概要
アプリケーションのデータをバックアップ・復元できます。

### データのエクスポート

1. フッターの **エクスポート** ボタンをクリック
2. JSONファイルがダウンロードされる
3. ファイル名: `mypage-data-YYYY-MM-DD.json`

**エクスポートされるデータ:**
- すべてのページのリンク
- セクション名
- TODOリスト
- メモ
- カレンダーイベント
- 設定情報（ナビゲーション、ポモドーロなど）

### データのインポート

1. フッターの **インポート** ボタンをクリック
2. ファイル選択ダイアログが開く
3. エクスポートしたJSONファイルを選択
4. データが復元される

**注意:**
- インポートすると、現在のデータが上書きされます
- バックアップを取ってから実行することを推奨

### 推奨バックアップ方法

1. **定期的なエクスポート**
   - 週に1回、または重要な変更後にエクスポート
   - クラウドストレージ（Dropbox、Google Driveなど）に保存

2. **複数バージョンの保持**
   - 過去数回分のバックアップを保持
   - ファイル名に日付が含まれているので管理しやすい

3. **ブラウザ移行時**
   - 新しいブラウザやデバイスでは、エクスポートしたファイルをインポート

### LocalStorageについて

このアプリケーションは、ブラウザのLocalStorageを使用してデータを保存しています。

**LocalStorageの特徴:**
- ブラウザごと、デバイスごとに独立
- オフラインでも動作
- ブラウザのキャッシュクリアで削除される可能性あり

**データ保存のキー:**
```
mainData             # メインページのリンク
projectData_<name>   # 各プロジェクトページのデータ
pomodoroSettings_global     # ポモドーロタイマー設定
pomodoroState_global        # ポモドーロタイマーの状態
navigationButtonSettings    # ナビゲーション設定
```
